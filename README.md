# Roto-Balancer
A Go-based simulation engine for Fantasy Baseball (Roto style).

## Features
- Calculates team totals for R, HR, RBI, SB, and AVG using Go maps and structs.
- Performance-based "Swap Analysis" to determine the marginal impact of adding free agents.
- Built with Go 1.24.

## Future Plans
- Load player data from JSON files.
- Integrate with live MLB Stats API.
- Add concurrency (Goroutines) for faster multi-player simulations.
